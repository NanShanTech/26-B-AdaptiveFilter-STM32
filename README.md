# B-ClearVoice

利用 STM32 内部 ADC1 进行 DMA 循环采样，确保实时性与低 CPU 占用。

- **采样参数**：采样率 $F_s = 40960\text{Hz}$，采样点数 $N = 8192$。

- **频率分辨率**：$\Delta f = \frac{F_s}{N} = 5\text{Hz}$。

  **波形识别**：分析基波与谐波（$3f$、$5f$）的幅值比，自动判断干扰信号 $B$ 的波形类型（正弦、方波或三角波） 。

## 2.2 幅值计算与能量补偿

系统采用“硬件检测+软件校验”的双重方案：

1. **干扰信号 $B$**：经外部有效值检测模块提取 $RMS_B$，根据 FFT 识别的波形类型推算其能量与峰峰值。
2. **有用信号 $A$**：
   - **频域提取**：根据 FFT 结果计算 $V_{p\_A} = \frac{2 \times Mag_A}{N}$。
   - **能量校验**：利用混合信号总能量与干扰能量之差进行修正，即 $RMS_A = \sqrt{RMS_C^2 - RMS_B^2}$。

## 2.3 信号重构与 PID 锁相 (PLL)

- **AD9910 输出**：将推算的有用信号频率与幅值下发至 AD9910 DDS 模块，生成纯净的重构信号。
- **相位跟踪**：对输出信号进行反馈采样，测量其与原信号的相位差 $\Delta \phi$。
- **PID 负反馈**：使用 PID 算法动态调节 AD9910 的相位控制字（POW），使系统达到同频同相显示，彻底解决波形漂移问题。

## 3. 硬件模块连接

1. **输入端**：信号 $C$ 和参考信号 $B$ 分别接入 ADC 通道和有效值检测模块。

2. **输出端**：AD9910 产生重构信号 $E$。
